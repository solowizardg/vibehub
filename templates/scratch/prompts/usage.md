# Scratch Template Usage

This is a **blank canvas** template with no pre-configured files.

## Requirements

The LLM must generate **ALL files from scratch**, including:

- `package.json` with correct dependencies
- Entry point (e.g., `index.html`, `main.js`, or `src/index.ts`)
- Configuration files for the chosen tooling (e.g., `vite.config.ts`, `tsconfig.json`)
- Any other files needed for a runnable project

## Guidelines

1. **Choose the most appropriate tech stack** based on user requirements
2. **Always include**:
   - `package.json` with correct dependencies and scripts
   - A valid entry point
   - Proper configuration files for the chosen framework/library
3. **Follow modern best practices** for whatever framework/library is chosen
4. **Include TypeScript configuration** (`tsconfig.json`) if using TypeScript
5. **Ensure the project is immediately runnable** after `npm install`

The project must be complete and executable with no missing pieces.
