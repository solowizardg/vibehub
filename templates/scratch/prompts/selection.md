# When to Use Scratch Template

Select the **scratch** template when:

- The user wants **full control** over project structure and technology choices
- The user specifies a **non-standard stack** or uncommon framework
- The user explicitly requests **vanilla JavaScript** or plain HTML/CSS/JS with no framework
- The user needs a framework or setup **not covered by other templates** (e.g., Svelte, Angular, Astro, Eleventy, or custom tooling)
- The user says "from scratch", "blank", "empty", or "no framework"
- No other template matches the user's requirements

This is the fallback template when no framework-specific template applies.
