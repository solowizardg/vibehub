# Template Selection - Next.js App

Use this template when the user wants:

- **SSR/SSG applications** - Server-Side Rendering or Static Site Generation for content that benefits from pre-rendering
- **Marketing sites** - Landing pages, product pages, or content-heavy sites that need fast initial loads
- **Full-stack apps with API routes** - Applications requiring both frontend pages and backend API endpoints in a single codebase
- **SEO-important pages** - Any project where search engine visibility and metadata control are critical (blogs, e-commerce product pages, documentation)
- **React applications** that need server components, streaming, or incremental static regeneration

Prefer another template when:

- The project is a client-only SPA with no SEO needs (consider a Vite-based template)
- The primary focus is static/client-only dashboards or tools with no server requirements
- You need Cloudflare Workers/Durable Objects as the backend (use a Worker template instead)
